## Backend

**Set up environment**

### Bước 1 — Tạo virtualenv và kích hoạt
    - Windows (PowerShell)
        python -m venv venv
        venv\Scripts\Activate

    - Linux / macOS
        python3 -m venv venv
        source venv/bin/activate

### Bước 2 — Cài đặt tất cả thư viện cần thiết
    pip install -r requirements.txt
